# Holberton School - Higher-Level Programming

This repository contains all the projects completed during the **Higher-Level Programming** curriculum at Holberton School. These projects focus on mastering high-level languages like **Python** and **JavaScript**, as well as relational databases using **SQL (MySQL)**.

The curriculum covers advanced concepts such as object-oriented programming (OOP) principles, data structures, exception handling, DOM manipulation, and database management.

## ‚öôÔ∏è General Requirements

All projects in this repository adhere to the following general standards:

  * **Environment:** Files are compiled and tested on **Ubuntu 20.04 LTS**.
  * **Python:** Scripts are written in **Python 3.8** and adhere to the **PyCode Style (PEP 8)**.
  * **JavaScript:** Scripts are written in **JavaScript (ES6)** and are generally tested in a modern browser environment.
  * **SQL:** All SQL scripts are executed on a **MySQL** server.
  * **Shebang:** Python and shell scripts include the correct shebang line (`#!/usr/bin/python3`).
  * **Documentation:** Functions, modules, and classes are documented with clear docstrings and comments.
